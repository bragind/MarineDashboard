# MarineDashboard
# 🚢 Marine Dashboard

Профессиональная кроссплатформенная цифровая приборная панель для морских судов. Проект разработан на **Qt 5.7** с целью последующей миграции на **Qt 6.x**.

Этот репозиторий демонстрирует не только разработку интерфейсов, но и процесс поддержки legacy-кода и стратегию миграции на современные версии фреймворка.

## 🎯 Цель проекта
Создать полностью функциональную морскую приборную панель в рамках 4-недельного интенсива.
**Стратегия обучения:**
1.  **Фаза 1 (Qt 5.7):** Построение устойчивой архитектуры, работа с ограничениями старой версии, фокус на стандартном C++14.
2.  **Фаза 2 (Qt 6.x):** Рефакторинг и портирование проекта на современный Qt, использование новых возможностей движка QML и C++17/20.

## ✨ Ключевые возможности
- **Приборы реального времени:** Скорость (узлы), Глубина, Компас, Ветер, Обороты двигателя, Уровень топлива, Тримы.
- **Симуляция данных:** Встроенный симулятор протоколов **NMEA 0183/2000**.
- **Адаптивный интерфейс:** Поддержка Windows, Linux и Android (Qt Quick Controls 2).
- **Темы оформления:** Переключаемые тёмная/светлая морские темы.
- **Система тревог:** Визуальные и логические алерты при критических состояниях судна.
- **Графики:** Интеграция Qt Charts для визуализации исторических данных.

## 🛠 Технологический стек (Фаза 1)
- **Язык:** C++14 (оптимально для Qt 5.7)
- **Фреймворк:** Qt 5.7 (Core, Gui, QML, Charts, Quick)
- **Система сборки:** CMake / qmake
- **Платформы:** Windows, Linux, Android

## 🏗 Архитектура и паттерны
Код написан с учётом будущей миграции, минимизируя зависимость от специфичных функций Qt 5:
- **MVC (Model-View-Controller)** — строгое разделение логики и UI.
- **Singleton** — для глобальных менеджеров.
- **Factory Pattern** — для абстракции создания приборов.
- **Observer Pattern** — через сигналы/слоты Qt.
- **PIMPL** — для скрытия реализации и ускорения компиляции.
- **CRTP** — для статического полиморфизма.
